What the theme represents

The public repository is not just a stylesheet or child-theme tweak. It contains a full WordPress theme codebase inside the `gravit` theme directory, including archive, page, single, search, 404, content-format, and custom page templates alongside the CSS.

That matters because it shows theme work as structural publishing work: template logic, page composition, typography, and content flow all shape what a site can be.

Why it matters in this archive

This artifact fits the Studio track because it sits where interface form and publishing structure become inseparable. A theme is not just a skin. It is a small publishing system with opinions about reading, layout, and what kinds of content are foregrounded.

The Gravit base also makes the design lineage legible: minimal, content-first, responsive, and typographically led rather than crowded with interface chrome.
